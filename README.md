# Distribuidora Búho 🦉

es una plataforma web diseñada para digitalizar la venta de bebidas al por mayor y menor. Permite a los clientes navegar por un catálogo dinámico, filtrar productos en tiempo real, gestionar un carrito de compras con precios diferenciados (unidad o fardo) y finalizar el pedido mediante un enlace directo a WhatsApp.

### 🔗 Enlace del repositorio
Puedes encontrar el código fuente y las últimas actualizaciones aquí:
[https://github.com/Nico30888/distribuidora.git]



## Instrucciones para ejecutarlo.
 
Sigue estos pasos para ver el proyecto funcionando en tu computadora:

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/Nico30888/distribuidora.git
    ```
2.  **Preparar las imágenes:**
    Asegúrate de que las fotos de los productos estén en la misma carpeta que los archivos de código y que sus nombres coincidan con los definidos en `distri.js`.
3.  **Abrir el proyecto:**
    Simplemente haz doble clic en el archivo `distri.html` para abrirlo en tu navegador.
    
## Tecnologías utilizadas
###### Para el desarrollo de este proyecto se utilizaron las siguientes herramientas y lenguajes:


**HTML:** Para la estructura y el contenido del sitio

**CSS:** Para el diseño visual, incluyendo el uso de variables, temas (oscuro/claro) y diseño responsivo.

**JS:**  Para toda la lógica del negocio, incluyendo:

    * Manipulación dinámica del DOM.
    * Gestión del estado del carrito.
    * Sistema de búsqueda y filtrado.
    * Google Fonts: Tipografía "Poppins" para que se vea moderno
